# SupplyAcustic

`CURSO: Análise e Desenvolvimento de Sistemas`

`DISCIPLINA: Trabalho Interdisciplinar Aplicações Inovadoras`

`1º semestre/2026`

O **SupplyAcustic** é uma plataforma web de análise acústica inteligente que utiliza inteligência artificial para diagnosticar e prever o comportamento sonoro de ambientes antes mesmo de sua construção ou reforma. A ferramenta permite que profissionais da arquitetura, engenharia e design avaliem parâmetros como tempo de reverberação (RT60), absorção sonora e distribuição de frequências, recebendo relatórios automatizados e sugestões de tratamento acústico.

A plataforma democratiza o acesso à consultoria acústica especializada — antes restrita a softwares caros e profissionais altamente especializados — tornando-a acessível via navegador, de forma intuitiva e com custo acessível.

## Instruções de utilização

A aplicação está disponível online e pode ser acessada pelo seguinte endereço:

URL: https://supplyacoustic.phflima.com

Não é necessário realizar nenhuma instalação local para utilizar as funcionalidades disponíveis ao usuário final, bastando acessar o sistema por meio de um navegador web compatível (Google Chrome, Microsoft Edge, Mozilla Firefox ou similares).
